# Betting Lessons Learned

Use this file to turn betting experience into better rules.

## Starting Lessons

- A small edge is only useful if the estimate is honest.
- A no-bet decision is a valid outcome.
- Casual discussion should not become ledger history.
- A model is most useful when it keeps emotion under control.
- Thin edges with weak data are easy to overrate.

## Repeated Mistakes To Watch For

- forcing action
- trusting gut over model
- overstating weak edges
- mixing cup logic into league matches
- logging discussion as if it were a real bet
