# Betting Discipline Checklist

Use this before every bet.

## Model Check

- [ ] I am using the correct model for the market.
- [ ] I used home and away splits where possible.
- [ ] I understand how the probability was estimated.

## Market Check

- [ ] I know the decimal odds.
- [ ] I know the implied probability.
- [ ] I know my estimated probability.
- [ ] The edge is at least `+3%`.

## Context Check

- [ ] If I am using a cup adjustment, Nic explicitly said the fixture is a cup.
- [ ] If not, I am using the league model by default.
- [ ] I am not forcing a bet because I want action.

## Staking Check

- [ ] I am using medium-risk staking: `0.5x Kelly`.
- [ ] The stake size fits the confidence level.

## Logging Check

- [ ] I will only log this if Nic writes the bet in the exact format ending with `- bet!`

## Final Decision

- [ ] If the edge is real and the process is clear, I can consider the bet.
- [ ] If the case depends mostly on instinct, I should pass.
