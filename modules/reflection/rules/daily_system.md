# Identity-Based Daily System

This file stores the current Reflection operating system for daily and weekly self-respect-based review.

## Philosophy

- Prioritize identity over outcomes.
- Measure success by self-respect and keeping promises.
- Core principle: "I don't like where I am yet, but I respect how I'm living."

## Morning

- Make bed
- Gratitude + blessing + prayer + visualization
- Ask: "What would Future Nic do today?"
- Define today's one win
- Morning walk
- Identity reminder

## Day

- Live intentionally
- Save before spending
- Follow fasting window
- No meals after `8pm`
- Learn for `1` hour

## Night

- `30` minutes of darkness + `4-4-6` breathing
- Manifestation + prayer
- Daily reflection:
  - Promise kept
  - Gratitude
  - Lesson learned
  - Did I behave like someone I respect?
  - Better tomorrow

## Reflection Framework

Judge days by:

- Character
- Integrity
- Consistency
- Promises kept

Do not judge days by:

- Income
- Validation
- Relationship status
- External achievements

## Weekly Review

Review each week through:

- Identity
- Health
- Finance
- Learning
- Relationships
- Weekly improvements

## Core Reminders

- Respect over comfort.
- Become someone Future Nic would be proud of.
- Small promises kept become a life transformed.
