# Lessons Learned

Use this file to turn experience into rules.

## Starting Lessons

- Support is a zone, not a magic number.
- Confirmation matters more than guessing.
- A clear risk plan reduces emotional decisions.
- Volatile, news-driven stocks deserve extra patience.
- Skipping a weak setup is a valid win.
- A planned trade is not the same thing as an executed trade.
- Only executed trades belong in `journal/trades.csv`.

## Repeated Mistakes To Watch For

- Buying too early
- Entering without confirmation
- Ignoring risk-to-reward
- Letting a swing trade become a hope trade
- Trading because I feel I need action

## Rules I Want To Reinforce

- Wait for structure
- Wait for confirmation
- Define the stop before entering
- Respect the stop if the setup breaks
