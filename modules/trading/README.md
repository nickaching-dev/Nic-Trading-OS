# Trading Module

This is the trading module inside NicOS.

Its purpose is to help Nic become a more disciplined swing trader through better process, clearer review, and structured learning.

## Current context

- Long-term ETF investing is separate from trading
- The `Moomoo` account is for learning swing trading
- Current learning focus:
  1. Market structure
  2. Support and resistance
  3. Risk vs reward
  4. Patience and confirmation

## Structure

- `context/`
- `rules/`
- `watchlist/`
- `journal/`
- `lessons/`
- `prompts/`
- `reports/`
- `templates/`

## Starter setup

`WOLF` is the current starter watchlist name in this module.

- Support: `45-47`
- First target: `53-55`
- Higher target: `58-60`
- Major resistance: `68-72`
- Do not buy support blindly
- Wait for bullish confirmation

## Important journal rule

Only add trades to `journal/trades.csv` if Nic explicitly says the trade was actually taken.
