# Investing Reference Sources

This file preserves the older low-hype educational source list for Nic's investing learning.

## Good Starting Points

- [Investor.gov](https://www.investor.gov/)
- [FINRA Investing Basics](https://www.finra.org/investors/investing/investing-basics)
- [Charles Schwab: New to Investing](https://www.schwab.com/learn/new-to-investing/)
- [Fidelity Learning Center](https://www.fidelity.com/learning-center/trading-investing/investing-for-beginners)
- [Khan Academy: Finance and Capital Markets](https://www.khanacademy.org/economics-finance-domain/core-finance)

## Filter Rule

Prefer sources that explain process, risk, and tradeoffs clearly. Avoid hype, urgency, screenshots, and guaranteed-return framing.
